SCADAsploit Framework
  • I’m the developer of “SCADAsploit”, an adversary simulation framework which includes dozens of auxiliary and exploits modules dedicated to OT systems, in particular PLC and SCADA mainly present on the Italian market. The database of modules allows to simplify the different phases of the pentest, such as network scanning, SCADA/PLC information gathering, upload/download application, read/write memory data and executing attacks like MitM, flooding, start/stop PLC, DoS and more.
